一、SSM整合流程

  1. 创建工程
  2. SSM整合
    • Spring
      • SpringConfig
    • MyBatis
      • MybatisConfig
      • JdbcConfig
      • jdbc.properties
    • SpringMVC
      • ServletConfig
      • SpringMvcConfig
  3. 相关功能模块的开发
    • domain（存放实体类）表与实体类
    • config （存放配置类） 配置类相关
    • dao（接口+自动代理） 数据访问层
    • service（接口+实现类）业务逻辑层
    • controller(代替servlet) WEB表现层

二、基本范例

  • 目录
  • pom.xml 依赖
  • JdbcConfig
  • MybatisConfig
  • ServletConfig
  • SpringConfig
  • SpringMvcConfig
  • BookController
  • Code
  • ProjectExceptionAdvice 用于捕捉并处理异常
  • Result
  • BookMapper ( BookDao )
  • Book（实体类）
  • BusinessException
  • SystemException
  • BookService
  • BookServiceImpl
  • jdbc.properties （放在目录的resource中）
  • BookServiceTest （关于测试类的一点东西）
